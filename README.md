## Output
The program prompts the user to enter a long text as input.  
After processing the text using Natural Language Processing techniques, it generates a concise summary by selecting the most important sentences.

### Example

Input:
Text summarization is a Natural Language Processing technique used to reduce long text into a short and meaningful summary. It helps users quickly understand the main idea of large documents. Text summarization is widely used in news applications, research papers, and search engines. Automatic summarization saves time and improves productivity.

Output:
Text summarization is a Natural Language Processing technique used to reduce long text into a short and meaningful summary. Text summarization is widely used in news applications, research papers, and search engines. Automatic summarization saves time and improves productivity.
